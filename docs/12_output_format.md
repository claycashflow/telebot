# 텔레그램 봇 /check 명령어 출력 형식

## 형식 설명

- 봇 이름: 지금이뉘
- 명령어: /check
- 타이틀: "저점인지 객관적으로 알아보기"

## 출력 형식 (템플릿)

```
지금이뉘 - 저점인지 객관적으로 알아보기
📊 {date} 장 마감 기준

🇰🇷 KOSPI {kospi_close} ▼ {kospi_change_pt}pt ({kospi_change_pct}%)

--------------------
① 52주 고점 대비 하락률 (MDD)
  - 52주 최고점: {역산값}
  - 현재: {kospi_close}
  - 하락률: {kospi_drawdown_pct}%

② 20일 이동평균 이격도
  - 20일 MA: {역산값}
  - 현재: {kospi_close}
  - 이격도: {disparity_20}

③ 전종목 20일 MA 하회 비율
  - 하회 종목: {below_ma20_ratio}%

--------------------
🔍 저점 판독 결과
  - 판정: {status}
  - 점수: {score}점
  - 근거:
  - {reason1}
  - {reason2}
  ...

🌐 매크로 상황
  - WTI: {wti}  /  Dubai: {dubai}
  - 미국 GDP YoY: {us_gdp_yoy}%
  - 고용 동향: {us_jobs}
  - 반도체 전망: {semiconductor_earnings_view}

📌 주목 업종
  - {sectors}

⚠️ 본 결과는 규칙 기반 판정 보조 도구이며 투자 자문이 아니다.
```

## 렌더링 예시 (샘플 데이터 기준)

```
지금이뉘 - 저점인지 객관적으로 알아보기
📊 2026-03-23 장 마감 기준

🇰🇷 KOSPI 2,612.34 ▼ 23.50pt (-0.89%)

--------------------
① 52주 고점 대비 하락률 (MDD)
  - 52주 최고점: 3,233.09
  - 현재: 2,612.34
  - 하락률: -19.2%

② 20일 이동평균 이격도
  - 20일 MA: 2,861.27
  - 현재: 2,612.34
  - 이격도: 91.3

③ 전종목 20일 MA 하회 비율
  - 하회 종목: 68.4%

--------------------
🔍 저점 판독 결과
  - 판정: 진 바닥 확인
  - 점수: 9점
  - 근거:
  - 하락률이 기술적 저점 후보 구간이다.
  - 이격도가 저평가 신호 구간이다.
  - 변동성이 높아 공포 확대로 해석할 수 있다.
  - 이동평균선 지지 신호가 있다.
  - 바닥 패턴 신호가 강화되고 있다.
  - 유가 부담이 있어 감점 요인이다.
  - 반도체 이익 전망이 완충 요인이다.

🌐 매크로 상황
  - WTI: 84.2  /  Dubai: 81.7
  - 미국 GDP YoY: 3.2%
  - 고용 동향: stable
  - 반도체 전망: positive

📌 주목 업종
  - 반도체

⚠️ 본 결과는 규칙 기반 판정 보조 도구이며 투자 자문이 아니다.
```

## 비고

- `52주 최고점`과 `20일 MA`는 입력값(`kospi_close`, `kospi_drawdown_pct`, `disparity_20`)으로 역산한다.
- 상승 시 `▲`, 하락 시 `▼`, 보합 시 `-`로 표시한다.
